# -kotlin-lab7
##  Description
Projet Kotlin pour le TP/Lab 7 — Nullability.
Ce lab montre la différence entre Student et Student?, l’utilisation de !! (qui peut provoquer un NullPointerException) et de l’opérateur Elvis ?: pour éviter le crash en fournissant une valeur par défaut.

##  Utilisation
Cloner le dépôt et exécuter la fonction main().
Le programme demande :

Un ID → affiche l’étudiant ou provoque un NPE si introuvable.

Un nom → affiche l’étudiant ou "the student is not found" si introuvable.
